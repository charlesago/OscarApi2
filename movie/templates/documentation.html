Documentation de l'API

Méthode: POST

Cette route permet de créer un nouvel élément.

URL

POST /create

Paramètres

Nom Type Description Requis
name string Le nom de l'élément à créer. Oui
description string Une description de l'élément. Non

Exemple de corps de requête

{
    "name": "Mon Élement",
    "description": "Ceci est une description de l'élément."
}

Réponse

En cas de succès, la réponse contiendra les détails de l'élément créé.

Code de statut: 201 Created
Exemple de réponse
{
    "id": 1,
    "name": "Mon Élement",
    "description": "Ceci est une description de l'élément."
}
